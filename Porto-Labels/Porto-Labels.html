Overview

The Porto-Labels component allows you to add customizable labels to interface elements, helping to highlight important information. You can customize the color, size, and style to fit different design needs. Porto defines the appearance and behavior of labels, while their implementation is managed through OpenContent, providing greater flexibility.

For more details, check out the Porto-Labels repository on GitHub or the Porto documentation.

Settings Properties

Setting Type Description
LabelColor Select Specifies the color of the label. Options include:
  • badge badge-default: Default
  • badge badge-success: Success
  • badge badge-info: Info
  • badge badge-warning: Warning
  • badge badge-danger: Danger
  • badge badge-dark: Dark
  • badge badge-primary: Primary
  • badge badge-secondary: Secondary
  • badge badge-tertiary: Tertiary
  • badge badge-quaternary: Quaternary
LabelSize Select Specifies the size of the label. Options include:
  • badge badge-sm: Small
  • badge: Default
  • badge badge-lg: Large

Notes

Note: Ensure that the component's properties are set properly to match the design and layout needs of your page. Here are some key points to consider:
  • Use LabelColor to select a predefined color style. For example, Success or Danger.
  • Use LabelSize to adjust the size of the label. Options include Small, Default, and Large.
  • Ensure that the selected styles align with your page's overall design for consistency and readability.
  • Preview your changes to verify that the labels appear as expected in different screen sizes and layouts.
