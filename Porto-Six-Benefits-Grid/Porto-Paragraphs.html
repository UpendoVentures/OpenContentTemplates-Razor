Porto-Paragraphs Documentation

Overview

The Porto-Paragraphs component allows you to create customizable paragraphs with adjustable margins, padding, and rich text formatting. This component is highly flexible and integrates seamlessly with OpenContent.

For more details, check out the Porto-Paragraphs repository on GitHub.

Usage Example

Properties

Property Data Type Description
Paragraph String The paragraph text to display. This field is required and supports rich text formatting.

Settings Properties

Property Data Type Description
MarginTop String Defines the top margin of the paragraph. Options include mt-0, mt-1, mt-2, mt-3, mt-4, mt-5, and mt-auto.
MarginBottom String Defines the bottom margin of the paragraph. Options include mb-0, mb-1, mb-2, mb-3, mb-4, mb-5, and mb-auto.
PaddingTop String Defines the top padding of the paragraph. Options include pt-0, pt-1, pt-2, pt-3, pt-4, pt-5, and pt-auto.
PaddingBottom String Defines the bottom padding of the paragraph. Options include pb-0, pb-1, pb-2, pb-3, pb-4, pb-5, and pb-auto.
