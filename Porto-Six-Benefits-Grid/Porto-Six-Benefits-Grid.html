Porto-Six-Benefits-Grid Documentation

Overview

The Porto-Six-Benefits-Grid component adds a template which leverages Bootstrap to provide a highly stylized way to display 6 of something - especially benefits of a product/service.

For more details, check out the Porto-Six-Benefits-Grid repository on GitHub.

Usage Example

Properties

Property Data Type Description
Module Title String The title of the module. Optional and not rendered if left empty.
Module Anchor String Used to create #anchor-links on the page (numbers, letters, hyphens only).
Key Benefits Array A list of six benefits. Each benefit includes:
  • Headline: The headline text for the benefit (required).
  • Description: A short description of the benefit (required).
  • Icon Class: A FontAwesome icon class for the benefit (required).

Settings

Settings Properties

Property Data Type Description
Margin Top Select Defines the top margin of the component. Options include mt-0, mt-1, mt-2, mt-3, mt-4, mt-5, and mt-auto.
Margin Bottom Select Defines the bottom margin of the component. Options include mb-0, mb-1, mb-2, mb-3, mb-4, mb-5, and mb-auto.
Padding Top Select Defines the top padding of the component. Options include pt-0, pt-1, pt-2, pt-3, pt-4, pt-5, and pt-auto.
Padding Bottom Select Defines the bottom padding of the component. Options include pb-0, pb-1, pb-2, pb-3, pb-4, pb-5, and pb-auto.
