Porto-Animations Documentation

Overview

The Porto-Animations component allows you to enhance your content with visually appealing animations. It leverages predefined CSS animation classes, such as fadeInUp, bounceIn, and rotateIn, to bring dynamic effects to your layout. This component is highly customizable, enabling you to define animations, include optional images, and display multiple paragraphs of text. The layout and bindings are managed through OpenContent, ensuring seamless integration and flexibility for your design needs.

For more details, check out the Porto-Animations repository on GitHub or the Porto documentation.

Properties

Property Data Type Description
Module Title String Used to define the title of the module. This title is displayed in the DNN administration interface to help identify the module.
Module Anchor String Used to create anchor links on the page. This allows users to link directly to the module using a unique identifier. Only letters, numbers, and hyphens are allowed.
Image Image An optional image to display alongside the animation. Example: /Portals/0/Images/logo2.png.

Property Data Type Description

  • fadeInLeft

  • bounceInRight

    Rotate Animations

  • rotateInUpRight

Notes

Note: Ensure that the Animation property is set to a valid animation class to achieve the desired effect. Review the available options in the Porto Animations Documentation. Additionally, verify that the component's properties, such as Image and Paragraphs, are configured correctly to match the design and layout needs of your page.
