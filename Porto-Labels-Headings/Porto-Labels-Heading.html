Porto-Labels-Headings Documentation

Overview

The Porto-Labels-Headings component allows you to display headings with customizable subheadings, colors, and sizes. This component is highly flexible and integrates seamlessly with OpenContent.

For more details, check out the Porto-Labels-Headings repository on GitHub or the Porto documentation.

Usage Example

Properties

Property Data Type Description
Module Title String Used to define the title of the module. This title is displayed in the DNN administration interface to help identify the module.
Module Anchor String Used to create anchor links on the page. This allows users to link directly to the module using a unique identifier. Only letters, numbers, and hyphens are allowed.
Heading Text String The main heading text for the component. This field is required.
SubHeading String The subheading text for the component. This field is required.

Settings

Settings Properties

Property Data Type Description
SubHeading Color String Defines the color of the subheading. Options include: Default, Success, Info, Warning, Danger, Dark, Primary, Secondary, Tertiary, Quaternary.
SubHeading Size String Defines the size of the subheading. Options include h1, h2, h3, and more.
