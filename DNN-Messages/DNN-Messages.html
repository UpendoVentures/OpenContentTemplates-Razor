DNN-Messages Documentation

Overview

The DNN-Messages component allows you to display customizable messages with different types (e.g., Info, Error, Success, Warning). It supports optional icons and rich text content for flexibility.

For more details, check out the DNN-Messages repository on GitHub

Usage Example

Properties

Property Data Type Description
Module Title String Used to define the title of the module. This title is displayed in the DNN administration interface to help identify the module.
Module Anchor String Used to create anchor links on the page. This allows users to link directly to the module using a unique identifier. Only letters, numbers, and hyphens are allowed.
Title String The title of the message. Optional and not rendered if left empty.
Content HTML (Wysihtml) The main content of the message. Supports rich text formatting.
Message Type Radio buttons Specifies the type of message. Options include:
  • Info: Informational message.
  • Error: Error or validation summary.
  • Success: Success message.
  • Warning: Warning message.
Icon String An optional FontAwesome icon class for the message. Example: fa-solid fa-info-circle. Find more icons at FontAwesome.

Notes

Note: Ensure that the Message Type is set correctly to match the intended purpose of the message. Use the Icon field to add visual emphasis to the message.
