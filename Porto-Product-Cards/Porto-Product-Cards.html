Porto-Product-Cards Documentation

Overview

The Porto-Product-Cards component allows you to display a list of products with customizable details such as name, description, pricing, links, and images. It supports featured products and responsive layouts that adapt to the number of columns specified.

For more details, check out the Porto-Product-Cards repository on GitHub.

Usage Example

Edit

Properties

Property Data Type Description
Products Array A list of products. Each product includes:
  • Product Name: The name of the product (required).
  • Product Description: A detailed description of the product (required).
  • Monthly Price: The monthly price of the product. If left blank, displays "Contact for Pricing".
  • Monthly Link: URL for the monthly pricing checkout.
  • Yearly Price: The yearly price of the product. If left blank, displays "Contact for Pricing".
  • Yearly Link: URL for the yearly pricing checkout (required).
  • Header Image: An image representing the product (required).
  • Is Featured: A checkbox to mark the product as featured. Ideally, only one product should be featured.

Settings

Settings Properties

Property Data Type Description
Number of Columns Select Defines the number of columns for displaying the products. Options range from 1 to 12.
Margin Top Select Defines the top margin of the component. Options include mt-0, mt-1, mt-2, mt-3, mt-4, mt-5, and mt-auto.
Margin Bottom Select Defines the bottom margin of the component. Options include mb-0, mb-1, mb-2, mb-3, mb-4, mb-5, and mb-auto.
Padding Top Select Defines the top padding of the component. Options include pt-0, pt-1, pt-2, pt-3, pt-4, pt-5, and pt-auto.
Padding Bottom Select Defines the bottom padding of the component. Options include pb-0, pb-1, pb-2, pb-3, pb-4, pb-5, and pb-auto.
