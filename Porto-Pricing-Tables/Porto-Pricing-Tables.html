Porto-Pricing-Tables Documentation

Overview

The Porto-Pricing-Tables component allows you to create customizable pricing tables with features, pricing details, and action buttons. It supports various configurations to match your design needs.

For more details, check out the Porto-Pricing-Tables repository on GitHub or the Porto documentation.

Usage Example

Edit

Properties

Property Description
ModuleTitle Defines the title of the pricing table module. Example: Our Pricing Plans.
ParallaxImage Specifies the URL of the parallax background image. Example: https://example.com/parallax.jpg.
Items A list of pricing table items. Each item includes:
  • Title: The title of the pricing plan. Example: Basic Plan. (Required)
  • Description: A short description of the pricing plan. Example: Perfect for individuals. (Optional)
  • Price: The price of the plan. Example: $19.99/month. (Required)
  • ActionButtonText: The text for the action button. Example: Sign Up. (Required)
  • MostPopular: Boolean to mark the plan as the most popular. Example: true or false.
  • ShowPopularRibbon: Boolean to display a "Most Popular" ribbon on the plan. Example: true or false.
  • Features: A list of features included in the plan. Each feature includes:
    • FeatureDescriptionText: The description of the feature. Example: Unlimited access to all features.. (Required)

Settings Properties

Property Description Options
Pricing Table Style Defines the visual style of the pricing table.
  • Style 1: A clean and modern style.
  • Style 2: A more detailed and feature-rich style.
  • Parallax: Includes a parallax background effect.
Pricing Table Size Defines the size of the pricing table.
  • Standard: Default size for the pricing table.
  • Small: A more compact version of the pricing table.

Views

Below are examples of the different styles and configurations available for the Porto-Pricing-Tables component. Each style is customizable and can be adapted to your design needs.

Four Plans: Displays four pricing plans in a row with a clean and modern layout.

Three Plans: Displays three pricing plans in a row, ideal for compact layouts.

Alternative Style: A visually distinct style with unique design elements for pricing plans.

Small Blocks: Compact pricing plans displayed as small blocks, suitable for minimalistic designs.

Parallax: Pricing plans with a parallax background effect for a modern and dynamic look.

Note:
  • Ensure that the PricingTableStyle property is set correctly to achieve the desired layout and functionality.
  • Use the Parallax style to add a dynamic background effect to your pricing tables.
  • For compact layouts, consider using the Small Blocks or Three Plans styles.
