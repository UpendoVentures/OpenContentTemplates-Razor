Porto-Image-Gallery Documentation

Overview

The Porto-Image-Gallery component allows you to create visually appealing image galleries for your layout. It supports customizable styles, grid layouts, and lightbox functionality to match your design needs.

For more details, check out the Porto-Image-Gallery repository on GitHub or the Porto documentation.

Properties

Property Data Type Description Options / Example
Module Title String Used to define the title of the module. This title is displayed in the DNN administration interface to help identify the module.
Module Anchor String Used to create anchor links on the page. This allows users to link directly to the module using a unique identifier. Only letters, numbers, and hyphens are allowed.
Groups Array A list of groups to categorize gallery items.
  • Item: A string representing the group name.
Items Array A list of items to display in the gallery.
  • Image: A string representing the URL of the image. Example: https://example.com/image.jpg.
  • GridWidth: Defines the width of the grid item. Options include:
    • col-md-12: Full width.
    • col-md-6: Half width.
    • col-md-4: One-third width.
    • col-md-3: One-fourth width.
    • ...and more (from col-md-1 to col-md-12).
    Default: col-md-12.
  • ProjectTitle: A string representing the title of the project. Example: My Awesome Project.
  • ProjectType: A string representing the type of project. Example: Photography.
  • Group: A string representing the group to which the item belongs. Example: Portfolio.
  • W2: A boolean indicating whether the item spans two columns in the grid. Options:
    • true: Spans two columns.
    • false: Default, spans one column.

Settings

Settings Properties

Property Data Type Description Options
Style Select Defines the layout style of the gallery.
  • Default: A standard gallery layout with sorting and filtering options.
  • Full Width: A gallery layout that spans the full width of the container.
  • Masonry 1: A masonry-style gallery layout with evenly spaced items.
  • Masonry 2: A masonry-style gallery layout with custom item sizes.

Views

Below are examples of the different styles and configurations available for the Porto-Image-Gallery component. Each style is customizable and can be adapted to your design needs.

Default: A standard gallery layout with sorting and filtering options.

Full Width: A gallery layout that spans the full width of the container, showcasing images in a clean and modern design.

Masonry 1: A masonry-style gallery layout with evenly spaced items. Perfect for showcasing projects with varying image sizes.

Masonry 2: A masonry-style gallery layout with custom item sizes, offering a dynamic and visually appealing design.

Note: Ensure that the Style property is set correctly in the settings to achieve the desired layout and functionality.

For more details, check out the Porto Image Gallery documentation.
