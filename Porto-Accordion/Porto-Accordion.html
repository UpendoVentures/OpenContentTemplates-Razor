Overview

The Porto-Accordion component allows you to organize content into collapsible sections, optimizing space usage and enhancing user experience. It offers extensive customization options, enabling you to modify the style, size, and appearance of the accordions to fit your interface design.

Porto provides different variations, such as accordions with icons, dark text, no background, no borders, and various sizes. While Porto defines the appearance and behavior, its implementation is managed through OpenContent for greater flexibility.

For more details, check out the Porto-Accordion repository on GitHub or the Porto documentation.

Properties

Property Data Type Description
Module Title Text Allows you to set the title of the accordion.

Settings

Settings Properties

Property Data Type Description
Choose the Accordion Color Dropdown list (Select) Allows you to set the accordion color by selecting one of the following values: Default, Primary, Quaternary, Secondary, or Tertiary.
Choose the Accordion Size Dropdown list (Select) Allows you to set the accordion size by selecting one of the following values: Default, Large, or Small.
Include the Background? Dropdown list (Select) Allows you to select a background color for the accordion.
Include the Borders? Dropdown list (Select) Allows you to select whether the accordion will have borders or not.

Notes

Note: The Porto-Accordion component is designed to organize content into collapsible sections, optimizing space and improving user experience. Ensure the following properties are configured correctly:
  • Module Title: Sets the title of the accordion (optional).
  • Items: Defines the list of panels or tabs within the accordion (required).
  • Title: Specifies the title of each tab (required).
  • Content: Sets the content of each tab, supporting HTML (required).
  • Icon: Adds an icon to each tab, using values from FontAwesome (optional).
Additionally, you can customize the accordion's appearance with settings such as color, size, background, and borders. Examples of configurations and styles are provided above.
