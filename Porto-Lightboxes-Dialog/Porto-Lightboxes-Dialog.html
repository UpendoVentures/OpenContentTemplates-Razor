Porto-Lightboxes-Dialog Documentation

Overview

The Porto-Lightboxes-Dialog component allows you to create interactive lightboxes and dialog boxes with customizable styles, animations, and forms. It supports various configurations to match your design needs.

For more details, check out the Porto-Lightboxes-Dialog repository on GitHub or the Porto documentation.

Properties

Property Data Type Description Options / Example
Module Title String Used to define the title of the module. This title is displayed in the DNN administration interface to help identify the module.
Module Anchor String Used to create anchor links on the page. This allows users to link directly to the module using a unique identifier. Only letters, numbers, and hyphens are allowed.
Button Text String Defines the text displayed on the button that opens the dialog. Open Dialog
Dialog Title String Defines the title of the dialog box. My Dialog Title
Dialog Text String Defines the main text content of the dialog box. This property is required. This is the content of the dialog.
href String Defines the URL or link associated with the dialog. https://example.com
Form Title String Defines the title of the form inside the dialog. Contact Us
Form SubTitle String Defines the subtitle of the form inside the dialog. We'd love to hear from you!
Form Data Array A list of form fields to display inside the dialog.
  • InputText: The label or placeholder for the input field. Example: Your Name.
  • FormDataType: The type of input field. Options include:
    • text
    • email
    • url
    • textarea
  • Row: Number of rows for the input field (for textareas). Options: 1 to 10. Default: 5.
  • GridWidth: Defines the width of the input field in the grid. Options include:
    • col-md-12: Full width.
    • col-md-6: Half width.
    • col-md-4: One-third width.
    • ...and more (from col-md-1 to col-md-12).
    Default: col-md-12.
Form Button Text1 String Defines the text for the first button in the form. Submit
Form Button Text2 String Defines the text for the second button in the form. Cancel
Color Form Button1 String Defines the color theme for the first button in the form.
  • btn-primary: Primary color.
  • btn-secondary: Secondary color.
  • btn-tertiary: Tertiary color.
  • btn-quaternary: Quaternary color.
Color Form Button2 String Defines the color theme for the second button in the form.
  • btn-primary: Primary color.
  • btn-secondary: Secondary color.
  • btn-tertiary: Tertiary color.
  • btn-quaternary: Quaternary color.

Settings

Settings Properties

Property Data Type Description Options
Style String Defines the style of the lightbox or dialog.
  • Animation: Lightboxes with animation effects.
  • Popup: Lightboxes for embedding external content like YouTube, Vimeo, or Google Maps.
  • Form: Lightboxes with forms for user input.
Animation String Defines the type of animation for the lightbox. This property is used when Style is set to Animation.
  • with zoom anim: Lightbox opens with a zoom animation.
  • with move anim: Lightbox opens with a move animation.
Popup String Defines the type of popup content. This property is used when Style is set to Popup.
  • youtube: Embeds a YouTube video.
  • vimeo: Embeds a Vimeo video.
  • gmaps: Embeds a Google Maps location.
Color String Defines the color theme for the lightbox or dialog.
  • primary: Primary theme color.
  • secondary: Secondary theme color.
  • tertiary: Tertiary theme color.
  • quaternary: Quaternary theme color.
