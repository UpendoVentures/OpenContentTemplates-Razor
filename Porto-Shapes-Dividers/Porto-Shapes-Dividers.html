Overview

The Porto-Shapes-Dividers component allows you to select an image and apply different styles to use it as a visual divider on the page. This helps enhance the structure and design of the interface with custom separators. Various shapes and positions can be configured to fit any design. Porto defines the appearance and behavior of the divider, while its implementation is managed through OpenContent, providing greater flexibility.

For more details, check out the Porto-Shapes-Dividers repository on GitHub or the Porto documentation.

Settings

Settings Properties

Setting Type Description
Style Select Specifies the style of the shape divider. Options include:
  • Style 1: A smooth wave-like shape with reverse alignment.
  • Style 2: Circular patterns with overlapping layers.
  • Style 3: A combination of curves and circles for a dynamic effect.
  • Style 4: A simple angled shape with a clean finish.
  • Style 5: A sharp triangular pattern for a modern look.
  • Style 6: Overlapping polygons with a layered effect.
  • Style 7: A zigzag pattern with sharp edges.
  • Style 8: A complex wave with multiple layers and depth.
  • Style 9: A smooth curve with a minimalistic design.
  • Style 10: A layered wave with a subtle gradient effect.
  • Style 11: A semi-circular arc with a clean finish.
  • Style 12: Angled layers with increased depth.
  • Style 13: Reverse angled layers for a unique perspective.
  • Style 14: Funnel-shaped layers for a focused design.
  • Style 15: A parallax effect with layered shapes.
  • Style 16: A dynamic wave with overlapping curves.
  • Style 17: A complex pattern with sharp and smooth transitions.
  • Style 18: A layered wave with a bold design.
  • Style 19: A horizontal animation with triangular shapes.
  • Style 20: A layered polygon with horizontal animation.

Notes

Note: Ensure that the component's properties are configured correctly to match the design and layout needs of your page. Below are some key considerations:
  • Use Style to select the desired shape divider. Each style offers a unique visual effect, such as waves, zigzags, or polygons. Refer to the Settings Properties section for detailed descriptions of each style.
  • Ensure the selected Image complements the chosen style. For example, high-contrast images work well with bold shapes like Style 5 or Style 18.
  • Preview your changes to verify that the shape divider aligns properly with your page's content and design.
  • Test the component on different screen sizes to ensure responsiveness and visual consistency across devices.

For additional guidance, refer to the Porto Shapes & Dividers Documentation.
