Porto-ExtensionList Documentation

Overview

The Porto-ExtensionList component allows you to create and manage a list of elements. Each element can include a title, description, category, image, and two action icons: one linking to the homepage and another for downloading related resources. This component is highly customizable.

For more details, check out the Porto-ExtensionList repository on GitHub

Edit

Properties

Property Data Type Description
ModuleTitle string Used to define the title of the module. This title is displayed in the DNN administration interface to help identify the module.
ModuleAnchor string Used to create anchor links on the page. This allows users to link directly to the module using a unique identifier. Only letters, numbers, and hyphens are allowed.
Extensions Array Title: The title of the extension. This field is required.
Description: A detailed description of the extension. This field is required.
Category: The category or categories of the extension. Options include:
  • Developer
  • Module
  • Prompt
  • Provider
  • Scheduled
  • User Experience
  • Open-Source
  • Commercial
This field is required.
Homepage: The URL of the extension's homepage. This field is required.
Download: The URL for downloading the extension. This field is required.
Documentation: An optional URL linking to the documentation for the extension.
Logo: The URL of the extension's logo. This field is required.
BadgeText: Optional text to display as a badge for the extension.

Settings

Settings Properties

Property Data Type Description
Columns Select Options include 2, 3, 4, 6
SortBy Select Defines the sorting criteria. Options include Title and Description.
Margin (top) Select Defines the top margin of the component. Options include mt-0, mt-1, mt-2, mt-3, mt-4, mt-5, and mt-auto.
Margin (bottom) Select Defines the bottom margin of the component. Options include mb-0, mb-1, mb-2, mb-3, mb-4, mb-5, and mb-auto.
Padding (top) Select Defines the top padding of the component. Options include pt-0, pt-1, pt-2, pt-3, pt-4, pt-5, and pt-auto.
Padding (bottom) Select Defines the bottom padding of the component. Options include pb-0, pb-1, pb-2, pb-3, pb-4, pb-5, and pb-auto.

Notes
