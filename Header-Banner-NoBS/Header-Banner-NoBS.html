Header-Banner-NoBS Documentation

Overview

The Header-Banner-NoBS component allows you to create customizable announcement banners with various styles, colors, and text alignment options. This component is highly flexible and integrates seamlessly with OpenContent.

For more details, check out the Header-Banner-NoBS repository on GitHub.

Usage Example

Properties

Property Data Type Description
ModuleTitle String The title displayed above the banner.
BannerId String The unique identifier for the banner.
BannerContent String The main content displayed inside the banner. This field is required.
BannerLink String The link associated with the banner content.

Settings Properties

Property Data Type Description
HeaderBackgroundColor String Defines the background color of the banner. Default is #CE3439.
HeaderTextColor String Defines the text color of the banner. Default is #FFFFFF.
HeaderTextType String Defines the text type for the banner. Options include h1, h2, h3, h4, h5, and p. Default is h3.
HeaderTextStyle String Defines the text style for the banner. Options include bold, italic, and underline.
TextAlign String Defines the text alignment for the banner. Options include left, center, and right. Default is center.
HeaderTextShadow Boolean Specifies whether to apply a shadow effect to the text. Default is false.
HeaderTextShadowValue String Defines the shadow value for the text. Default is 2px 2px 4px rgba(0, 0, 0, 0.5).
WithCloseButton Boolean Specifies whether the banner includes a close button. Default is false.
