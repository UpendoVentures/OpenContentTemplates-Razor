Porto-Lightboxes Documentation

Overview

The Porto-Lightboxes component allows you to create visually appealing lightboxes with customizable styles, hover effects, zoom options, and more. It supports various configurations to match your design needs.

For more details, check out the Porto-Lightboxes repository on GitHub or the Porto documentation.

Usage Example

Edit

Properties

Property Data Type Description Options / Example
Module Title String Used to define the title of the module. This title is displayed in the DNN administration interface to help identify the module.
Module Anchor String Used to create anchor links on the page. This allows users to link directly to the module using a unique identifier. Only letters, numbers, and hyphens are allowed.
Single Image Image (upload & autocomplete) Defines the URL of a single image to display in the lightbox. https://example.com/image.jpg
Image Gallery List A list of images to display in a gallery lightbox.
  • Image: The URL of an image in the gallery. Example: https://example.com/gallery-image1.jpg.

Settings

Settings Properties

Property Data Type Description Options / Example
SHOW String Defines the type of lightbox to display.
  • Single-Image: Displays a single image in the lightbox.
  • Image-Gallery: Displays a gallery of images.
  • Image-GalleryCarousel: Displays a gallery of images in a carousel format.
DEFAULT WITH BORDER Boolean Specifies whether the lightbox should have a border.
  • true: Lightbox includes a border.
  • false: Lightbox does not include a border.
WITH ICON Boolean Specifies whether the lightbox should include an icon overlay.
  • true: Lightbox includes an icon overlay.
  • false: Lightbox does not include an icon overlay.
HOVER EFFECT Boolean Specifies whether the lightbox should include a hover effect.
  • true: Hover effect is enabled.
  • false: Hover effect is disabled.
ZOOM Boolean Specifies whether the lightbox should include a zoom effect.
  • true: Zoom effect is enabled.
  • false: Zoom effect is disabled.
